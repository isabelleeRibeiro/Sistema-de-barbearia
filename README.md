Esse código simula um sistema de gestão de barbearia: inicialmente, são criadas todas as funções que serão posteriormente chamadas
(qnt_serviços para a quantia de serviços prestados; total_faturado para o total dos valores dos serviços prestados; num
para a criação da fila de clientes; clientes, para armazenar os nomes dos clientes. Para cada cliente adicionado, um número
será adicionado à fila e, logo após todos os clientes serem cadastrados, o sistema exibirá a fila atual e passará para 
sua "segunda parte": a escolha de serviços. Para cada cliente, será escolhido 1 ou mais opções dentre cabelo, bigode e barba, 
cada uma com seu determinado valor, com essa sessão também contendo uma opção para finalizar os serviços. Após a finalização do 
serviço de um cliente, será passado para o próximo, até a finalização de todos. Com a finalização do serviços de todos os clientes, 
o "break" finaliza a sessão e o sistema exibe a quantidade de serviços prestados no total e o valor faturado total.
